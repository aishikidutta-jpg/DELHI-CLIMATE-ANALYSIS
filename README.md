# Analysing meteorological changes over Delhi 
This project is based on a dataset available on kaggle :[CLIMATE DATASET](https://www.kaggle.com/datasets/yug201/daily-climate-time-series-data-delhi-india)    
The user can download the csv file of the dataset and view it on their computer on MS excel (preferably) or other related apps.  
The code has been written on MATLAB Live Editor and has 9 visual graphs each for heatmap, humidity , pressure , rainfall, temperature range etc.   
its mostly a visual analysis from which inference can be drawn.
